# Operations

## Mathematical operations

The common basic math ops are available, including order of operation.

### Infix

#### Math

```aat
+ // addition
- // subtraction
* // multiplication
/ // division
% // modulo (remainder)
```

#### Bitwise

```aat
| // bitwise OR
& // bitwise AND
^ // bitwise XOR
```

### Boolean

#### Comparison

```aat
== // Equality
!= // Inequality
>  // Greater
>= // Greater than or equals
<  // Lower
<= // Lower than or equals
```

#### Composition

```aat
|| // boolean OR
&& // boolean AND
```

### Prefix

```aat
! // boolean negation
- // numeric negation
& // reference / address of
* // dereference
```
