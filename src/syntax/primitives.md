# Primitive Types

## Primitive numeric types

### Signed

```aat
i8
i16
i32
i64
```

### Unsigned

```aat
u8
u16
u32
u64
```

### Floating

```aat
f32
f64
```

## String and char

```aat
str
char
```

## Other

```aat
bool
type*     // pointer to `type`
&type     // reference to `type`
[type]    // slice of `type`
[type, n] // array of `type` with `n` length
```
